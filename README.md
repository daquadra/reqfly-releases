# ReqFly Releases

Repositório oficial para releases do aplicativo **ReqFly**.

## Sobre o ReqFly

ReqFly é uma aplicação disponível em [https://reqfly.mscodify.dev.br/](https://reqfly.mscodify.dev.br/)

## Releases

As versões do aplicativo são disponibilizadas através das [Releases](../../releases) deste repositório.

### Download

Para baixar a versão mais recente do ReqFly:

1. Acesse a seção [Releases](../../releases)
2. Selecione a versão desejada
3. Faça o download do arquivo apropriado para seu sistema operacional

### Instalação

Instruções específicas de instalação estão disponíveis em cada release.
